## GitHub Actions for Educators

GitHub Actions is GitHub's built-in automation platform that lets you build, test, and deploy code right from your repository. For educators, Actions is a powerful tool for autograding student assignments, deploying student projects to the web, and teaching real-world CI/CD workflows — all without leaving GitHub.

**Learning Objectives:**
- Explain what GitHub Actions is and how workflows, jobs, and steps relate to each other
- Create a basic workflow that runs automatically on push or pull request
- Use GitHub Actions to autograde student assignments with GitHub Classroom
- Deploy a student project to GitHub Pages using a custom Action
- Find and use pre-built Actions from the GitHub Marketplace

**Prerequisites:** Completion of the Git and GitHub Foundations Learning Journeys or a basic understanding of repositories, commits, and pull requests

**Sample Classroom Projects**
- Set up autograding for a Python assignment using [GitHub Classroom's built-in test runner](https://docs.github.com/en/education/manage-coursework-with-github-classroom/teach-with-github-classroom/use-autograding)
- Have students deploy a personal portfolio site to GitHub Pages using the [pages-action](https://github.com/actions/deploy-pages) — connects Markdown, HTML, and CI/CD in one assignment
- Create a workflow that runs a linter on student pull requests so they get instant code quality feedback before peer review
